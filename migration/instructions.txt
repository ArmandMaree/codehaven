+-----------------------------------------------------------------+
|                          INITIAL SETUP                          |
+-----------------------------------------------------------------+
1) Make sure sequelize-mig is installedd globally by running:
  npm install sequelize-mig -g

+-----------------------------------------------------------------+
|                   CREATE A NEW DATABASE TABLE                   |
+-----------------------------------------------------------------+
1) Create a new model by running
  npx sequelize-cli model:generate --name <TableName> --attributes id:number,name:string,deletedAt:date

+-----------------------------------------------------------------+
|                UPDATE AN EXISTING DATABASE TABLE                |
+-----------------------------------------------------------------+
1) Modify the model (this is located in src/migrations/models)

+-----------------------------------------------------------------+
|             MIGRATE DATABASE TO CONTAIN NEW CHANGES             |
+-----------------------------------------------------------------+
1) Then generate the migration file by running
  sequelize-mig migration:make -n <migration name>
2) Then you can upgrade the DB by running
  npx sequelize-cli db:migrate
  2.1) You can rollback the upgrade by running:
    npx sequelize-cli db:migrate:undo
  2.2) The environment that gets upgraded is specified by the NODE_ENV variable in the .env file
