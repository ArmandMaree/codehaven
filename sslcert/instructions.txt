1. Download SSL certificate from domain.com
2. Using FileZilla, place the .crt and .key files in /opt/bitnami/apache/conf/bitnami/certs
3. Then rename the files to server.crt and server.key
4. In order to automatically redirect HTTP traffic to HTTPS do this: https://docs.bitnami.com/bch/apps/wordpress/administration/force-https-apache/
5. Then restart Apache using this command. This might take a couple of minutes.
   sudo /opt/bitnami/ctlscript.sh restart apache
